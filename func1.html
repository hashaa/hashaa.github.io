Open sets and topology

1.1 Deﬁnition and examples

Deﬁnition: Let $X$ be a set and $T$ be a family of subsets of $X$. $T$ is called a topology on $X$ if:
  1. The empty set $\emptyset$ and $X$ are elements of $T$. 
  2. Arbitrary (ﬁnite or inﬁnite) unions $\cup_\alpha U_\alpha$ of elements $U_{\alpha}$of T belong to $T$ (or equivalently, $T$ is stable by arbitrary unions).
  3. Any ﬁnite intersection $$ of elements of T is in T (or equivalently, T is stable by ﬁnite intersection).

 Deﬁnition Set $X$ with a given topology deﬁned on it, i.e. the pair $(X, T)$, is a topological space.
Deﬁnition Elements of $T$ are called open sets.
To specify a topological space, it means to deﬁne a set $X$ and a topology in $X$, i.e., to indicate which subsets of $X$ are considered as open sets. Clearly, we can deﬁne on $X$ various diﬀerent topologies and therefore obtain diﬀerent topological spaces constructed on the same set $X$.
Any set $X$ can always be considered as a topological space:
  • the trivial topology $T_t = \{\emptyset, X\}$; 
  • the discrete topology $T_d =\{2^X,X\}$.
We notice that in the case of the discrete topology every subset of $(X, T_d)$ is open.
  1. The set $N$ is called a neighborhood of a point $x\in (X, T)$ if there exists an open set $O\in T$ such that $x\in O$ and $\O\in N$. The set of neighborhoods of $x$ is noted $N(x)$.
  2.  A point x ∈ (X, T) is called a contact point of a set $A$ if every neighborhood of $x$ contains at least one point of $A$;
  3.  A point $x\in (X, T)$ is called a limit point of a set $A$ if every neighborhood of $x$ contains at least one point of A diﬀerent of $x$.
  4. The set of all contact points of a set $A$ is called the closure of $A$, denoted by $\bar{A}$
  5. The largest open set contained in a given set $A$ is called the interior of $A$.
Theorem 1 Let $A$ be a subset of a topological space $(X, T)$. Then
  1. $ A\subset \bar{A}$,
  2. $\bar{\bar{A}} = \bar{A}$,
  3. if $B\subset A$, then $B\subset \bar{A}$,
  4.  for all $A, B \in X$, $\bar{A \cup B} =\bar{ A}\cup\bar{B}$.

1.2 Comparison of topologies

Deﬁnition Let $T_1$ and $T_2$ be two topologies deﬁned in the same set $X$. Then the topology $T_1$ is stronger than the topology $T_2$ if $T_2 \subset T_1$, i.e., if every set of the system $T_2$ is in a set of the system $T_1$.

Corollary 1 Let $A$ be any system of subsets of a set $X$. Then there exists a minimal topology in $X$ containing $A$, i.e., a topology $T(A)$ containing A and contained in every topology containing $A$.

Deﬁnition Let $\mathcal{A}$ be a system of subsets of $X$ and $A$ a ﬁxed subset of $X$. Then the system $\mathcal{A}_A$ consisting of all subsets of $X$ of the form $A\cap B$, $ B\in \mathcal{A}$ is called the trace of the system $\mathcal{A}$ on the set $A$.
